# Project: Google Workplace Microlearning Course

## Agents
- **adult-training-course-writer** — Primary agent for course creation (Mode A) and marketing collateral (Mode B)
- **industry-research-expert** — Research agent for market/industry analysis

## Workflow
1. Use `industry-research-expert` to research the topic first
2. Use `adult-training-course-writer` in Mode A to create course modules one at a time
3. Use `adult-training-course-writer` in Mode B only after all modules are approved
