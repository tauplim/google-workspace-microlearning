## Applied Bridge: Google Workspace Case

### Section A: Extract Target Learner Profile

| Bridge Field | Extracted from Research |
|-------------|------------------------|
| **Primary learner role** | Mid-market professionals (300-1000 employees) who use Google Workspace daily but underutilize AI features |
| **Experience level** | Basic to intermediate — familiar with core apps but AI/Gemini is new |
| **Time constraints** | 13.7 hours/year average learning budget — must be efficient, bite-sized |
| **Pain points** | (1) Can't measure training ROI; (2) 21 days/year productivity gap not solved by software alone |
| **Success definition** | Immediate productivity gains; ROI becomes visible within 30 days |

### Section E: Success Metrics Definition

| Metric Type | Definition |
|-------------|------------|
| **Immediate** | Learner can execute one Gemini-powered workflow after each 30-minute session |
| **Short-term (30 days)** | Productivity improvement visible; ROI measurement possible |
| **Long-term** | Address 21-day productivity gap; sustained behavior change |

### Section F: Module Priority Guidance

| Priority | Topic | Rationale |
|----------|-------|-----------|
| **Highest** | Foundation (Gemini + core apps) | Must establish baseline AI familiarity |
| **Highest** | AI integration across apps | Primary differentiator vs competitors |
| **Highest** | Automation/prompt chaining | Addresses 21-day productivity gap directly |
| **Secondary** | Collaboration with AI | Core Workspace value proposition |
| **Secondary** | File management/search | Common pain point; underutilized features |
| **Tertiary** | Advanced features (Vault, etc.) | Niche; not for mid-market primary focus |
| **Exclusions** | Non-Google tools integration | Stay focused on native Workspace |

---

### Section G: Differentiation Focus

1. **Native Google + Gemini integration** — No third-party wrapper; direct AI access within tools learners already use
2. **Prompt chaining for compound workflows** — Turn single prompts into automated sequences (addresses 21-day gap)
3. **Microlearning with compression artifacts** — Bite-sized + practical takeaway per session (addresses 13.7 hr constraint)

## Summary for Course Writer

**Target:** Mid-market professional, 13.7 hrs/year learning budget, wants ROI and productivity

**Differentiation:** Native Google AI integration + microlearning format + automation focus

**Must deliver:** Immediate productivity gains visible within 30 days; ROI measurement patterns

**Key constraint:** 30-minute sessions; modules may span multiple sessions
