Natural Language Search Examples

"Find contracts I haven't reviewed yet"

→ Locates unreviewed contract files

"Where is the presentation from last quarter's all-hands?"

→ Shows presentation with metadata

"Find files modified by Sarah in the past week"

→ Filters by owner and date

"Show me all spreadsheets about budget"

→ Semantic search across content

WORKSHOP

Task 1: AI-Assisted File Search

Time: 3 minutes

Try these natural language searches in Drive:

Search 1:

"Find files I haven't opened in over a month"

Search 2:

"Find [topic from your work]"

Search 3:

"Show me files modified this week"

WORKSHOP

Task 2: Create Project Folder Structure

Time: 4 minutes

Create a project folder with this structure:

[Your Project]/

├── 01-Background/

├── 02-Materials/

├── 03-Drafts/

├── 04-Feedback/

└── 05-Archive/

Add brief descriptions to each folder. Move at least one file into the structure.

WORKSHOP

Task 3: Set Sharing Permissions

Time: 3 minutes

1

Share main project folder with a partner

2

Choose "Anyone with link can view"

3

For a subfolder, choose "Restricted - only you"

Pattern Recap

"AI search means 'what' over 'where.' But good folder structure still helps when AI isn't available."

Drive Organization Principles:

1

Shallow hierarchy (max 3-4 levels)

2

Numbered folders (01, 02, 03)

3

Descriptive names with dates

4

Consistent structure across projects

Take-Away: Drive Organization System

PROJECT FOLDER TEMPLATE

MASTER PRINCIPLES:

Number folders • Create Archive • Use YYYY-MM-DD dates • Star critical files • Review monthly

Standard Project Structure:

Project [Name] YYYY/

├── 00-Admin/ ← Contracts, setup

├── 01-Research/ ← Background

├── 02-Materials/ ← Assets, data

├── 03-Drafts/ ← Working docs

├── 04-Review/ ← Feedback

├── 05-Final/ ← Approved

└── 06-Archive/ ← Completed
