The 5 Core Applications

Gmail

Email communication

Summarize threads, draft replies

Calendar

Scheduling

Find times, generate agendas

Drive

File storage

Natural language search

Docs/Sheets/Slides

Documents

Write, analyze, design

Meet

Video conferencing

Meeting notes, transcription

The Workspace Advantage

Unlike standalone tools, Google Workspace apps work together seamlessly

Single Sign-On

One identity across all apps. No multiple passwords.

Central Storage

Files stored centrally in Drive. Access from any app.

Auto AI Updates

New AI features added automatically across all apps.

Cross-Device Sync

Seamlessly switch between desktop, tablet, and phone.

Where to Find Gemini

Gemini is embedded directly in each Workspace app

Application Gemini Location Activation
Gmail Sparkle icon in compose window Click to activate
Docs Tools menu → Gemini OR shortcut Ctrl+Shift+M
Sheets Sparkle icon in toolbar Click to activate
Slides Insert → Gemini menu Click to activate
Drive gemini.google.com Web interface
Calendar Scheduling assist panel Right sidebar
Meet Meeting notes (post-meeting) Auto-generated

Three Ways Gemini Helps

Generate

Create content from scratch — drafts, emails, ideas, reports

Summarize

Condense long documents, email threads, and meetings into key points

Analyze

Extract insights from data, identify patterns, compare options

Live Demonstration

Watch the trainer perform these actions

Gmail Summarization

  1. 1. Open a long email thread
  2. 2. Click Gemini sparkle icon
  3. 3. Select "Summarize this email"
  4. 4. Observe AI-generated summary

Docs Generation

  1. 1. Open new Google Doc
  2. 2. Press Ctrl+Shift+M
  3. 3. Type your prompt
  4. 4. Observe AI draft content

Sheets Formula

  1. 1. Open spreadsheet
  2. 2. Click Gemini in toolbar
  3. 3. Ask for formula help
  4. 4. Observe AI-generated formula

Drive Search

  1. 1. Open gemini.google.com
  2. 2. Type natural language query
  3. 3. Ask to find a specific file
  4. 4. Watch AI locate the file
