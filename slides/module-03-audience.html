The Repetition Problem

Professionals spend significant time recreating similar documents

Document Type Frequency Time to Create
Meeting notes Daily 5-10 min
Status updates Weekly 15-20 min
Project briefs Per project 30-45 min
Client emails Multiple/day 5-10 min each

65 hours/year

Time lost creating 5 similar documents/week at 15 min each

The Template Solution

Fixed Elements

Already written and consistent:

  • Headers and section titles
  • Standard text and instructions
  • Formatting and structure
  • Branding and logos

Variable Elements

Marked as placeholders:

  • [Client Name]
  • [DATE]
  • [AGENDA ITEMS]
  • [ACTION ITEMS]

AI Enhancement: Gemini fills placeholders AND generates appropriate content

Anatomy of an AI-Friendly Template

TEMPLATE COMPONENTS

HEADER (Fixed)

[Company Logo] | Document Type: [TYPE] | Date: [DATE]

CONTENT (Variable)

## [TOPIC/HEADER]

[Variable content - AI fills this]

## Key Points

• [Point 1 - AI generates]

• [Point 2 - AI generates]

FOOTER (Fixed)

Author: [YOUR NAME] | Version 1.0 | [DATE]

Variable Marker Conventions:

[BRACKETS] = Replace | [UPPER CASE] = Auto-fill | [TASK/ACTION] = AI generates

Effective vs. Weak Prompts

Weak Prompt

"Write meeting notes"

Too vague - AI doesn't know context, audience, or specifics

Strong Prompt

"Write meeting notes for a 30-minute standup with 5 team members. Include: attendance, yesterday's accomplishments, today's priorities, blockers."

Specific context = accurate, useful output

WORKSHOP

Task 1: Build Your Meeting Notes Template

Time: 5 minutes

Create this structure:

MEETING NOTES TEMPLATE

Date: [DATE] | Type: [TYPE] | Attendees: [NAMES]

AGENDA

1. [AGENDA ITEM 1]

2. [AGENDA ITEM 2]

DISCUSSION NOTES [AI fills]

KEY DECISIONS • [DECISION 1]

ACTION ITEMS ☐ [ACTION] - Owner: [NAME]

NEXT MEETING: [DATE]

Use Heading 2

WORKSHOP

Task 2: Fill Template Using AI

Time: 5 minutes

  1. 1

    Make a copy of your template

  2. 2

    Open Gemini: Ctrl+Shift+M

  3. 3

    Paste the prompt below

Fill this meeting notes template for:

• Date: May 27, 2026

• Meeting Type: Weekly Team Standup

• Attendees: Alex, Jordan, Taylor, Morgan

• Generate realistic notes, 2 decisions, 3 action items

VARIATION EXERCISE

Generate Content Variations

Time: 3 minutes

Take your filled "Key Decisions" section and generate 3 variations:

1. Formal Executive

"Rewrite as formal executive briefing"

2. Bullet Summary

"Summarize as 2-bullet update"

3. Social Media

"Convert to social post (100 words)"

Which tone was most useful? When would you use each?

Pattern Recap

"Templates + AI = Consistent documents, generated instantly. Build once, use forever, fill with AI."

Template Building Rules:

1

Mark every variable with [BRACKETS]

2

Give AI context (who, what, when, why)

3

Preview AI output before accepting

4

Save each output with meaningful filename
