## LEARN Phase

### Part 1: Why Templates Save Time (3 minutes)

**The Repetition Problem**

Professionals spend significant time recreating similar documents:

| Document Type | Frequency | Time to Create |
|---------------|-----------|----------------|
| Meeting notes | Daily | 5-10 min |
| Status updates | Weekly | 15-20 min |
| Project briefs | Per project | 30-45 min |
| Client emails | Multiple/day | 5-10 min each |

**Time Cost:** If you create 5 similar documents/week at 15 minutes each, that's 65 hours/year.

**The Template Solution**

A template is a **pre-structured document** where:
- **Fixed elements** are already written (headers, sections, standard text)
- **Variable elements** are marked as placeholders (client name, dates, specifics)

**AI enhancement:** Gemini can fill placeholders AND generate appropriate content for each variable.

---

### Part 2: Template Structure with AI Variables (4 minutes)

**Anatomy of an AI-Friendly Template:**

```
┌─────────────────────────────────────────────────────────────┐
│ TEMPLATE COMPONENTS                                          │
├─────────────────────────────────────────────────────────────┤
│                                                             │
│  HEADER SECTION (Fixed)                                      │
│  ─────────────────────                                      │
│  [Company Logo]                                              │
│  Document Type: [TYPE]              Date: [DATE]            │
│                                                             │
│  CONTENT SECTIONS (Variable)                                │
│  ─────────────────────────                                  │
│  ## [TOPIC/HEADER]                                          │
│  [Variable content goes here - AI fills this]               │
│                                                             │
│  ## Key Points                                              │
│  • [Point 1 - AI generates]                                 │
│  • [Point 2 - AI generates]                                 │
│  • [Point 3 - AI generates]                                 │
│                                                             │
│  FOOTER (Fixed)                                             │
│  ─────────────────────                                      │
│  Author: [YOUR NAME] | Version 1.0 | [DATE]                 │
│                                                             │
└─────────────────────────────────────────────────────────────┘
```

**Variable Marker Conventions:**

| Marker Type | Example | AI Instruction |
|-------------|---------|----------------|
| `[BRACKETS]` | `[Client Name]` | Replace with specific value |
| `[UPPER CASE]` | `[DATE]` | Auto-fill from context |
| `[TASK/ACTION]` | `[STATUS UPDATE]` | AI generates appropriate content |

**Effective Variable Instructions:**

**❌ Weak:** "Write meeting notes"
**✅ Strong:** "Write meeting notes for a 30-minute standup with 5 team members. Include: attendance, yesterday's accomplishments, today's priorities, blockers."

### Part 3: Live Demonstration (4 minutes)

**Watch the trainer perform these actions:**

1. **Open Template Folder**
   - Navigate to pre-shared Drive folder
   - Open "Meeting Notes Template"

2. **Show Template Structure**
   - Highlight placeholder markers
   - Explain fixed vs. variable sections

3. **Fill Template with AI**
   - Click Gemini in Docs
   - Prompt: "Fill this template for a team standup on May 27, 2026. Team name: Product Development. Members attending: Sarah, Mike, Lisa."
   - Show AI-generated content

4. **Generate Variations**
   - Take one section (e.g., "Key Accomplishments")

     - "Casual team update"

5. **Save as New Document**
   - File → Make a copy
   - Rename with date and context
   - Demonstrate template preservation

## CONSOLIDATE Phase

### Guided Workshop: Build a Template (10 minutes)

**Task 1: Create Your Meeting Notes Template (5 minutes)**

1. Create new Google Doc titled: "My Meeting Notes Template"
2. Add these sections:

```
[Your Template Below - Build as you go]

───────────────────────────────────────────

MEETING NOTES TEMPLATE

Date: [DATE]
Meeting Type: [TYPE - e.g., Standup, Review, 1:1]
Attendees: [NAMES]

AGENDA
1. [AGENDA ITEM 1]
2. [AGENDA ITEM 2]
3. [AGENDA ITEM 3]

DISCUSSION NOTES
[AI will fill based on meeting context]

KEY DECISIONS
• [DECISION 1]
• [DECISION 2]

ACTION ITEMS
☐ [ACTION 1] - Owner: [NAME] - Due: [DATE]
☐ [ACTION 2] - Owner: [NAME] - Due: [DATE]

NEXT MEETING
[DATE AND TIME]
```

3. Format with headings (Heading 2 for sections)
4. Bold the placeholders in [BRACKETS]
5. Save to your Drive

**Task 2: Fill Template Using AI (5 minutes)**

1. Make a copy of your template
2. Open Gemini (Ctrl+Shift+M)
3. Paste this prompt:

```
Fill this meeting notes template for:
- Date: May 27, 2026
- Meeting Type: Weekly Team Standup
- Attendees: Alex, Jordan, Taylor, Morgan
- Agenda: Project updates, blockers, team recognition

Generate realistic discussion notes, at least 2 decisions, and 3 action items.
```

4. Review AI-generated content
5. Adjust any mismatches

**Task 3: Save and Document (2 minutes)**

1. Save as: "Standup Notes - 2026-05-27"
2. Add to the template folder
3. Note what worked well about the AI output

### Variation Generation Exercise (3 minutes)

**Take your filled document and generate variations:**

1. Select the "Key Decisions" section
2. Ask Gemini: "Rewrite this in a formal executive briefing style"
3. Ask Gemini: "Summarize this as a 2-bullet update"
4. Ask Gemini: "Convert this to a social media announcement (100 words)"

**Record your favorite variation:**
- Which tone/style was most useful?
- When would you use each variation?

## COMPRESS Phase

### Pattern Recap (1 minute)

**The ONE thing to remember:**

> "Templates + AI = Consistent documents, generated instantly. Build once, use forever, fill with AI."

**Template Building Rules:**
1. Mark every variable with [BRACKETS]
2. Give AI context (who, what, when, why)
3. Preview AI output before accepting
4. Save each output with a meaningful filename
