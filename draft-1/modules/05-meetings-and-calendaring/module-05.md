## LEARN Phase

### Part 1: Meeting Inefficiency Costs (3 minutes)

**The True Cost of Meetings**

Meetings consume significant professional time:

| Metric | Data |
|--------|------|
| Average meetings per week | 8-12 for knowledge workers |
| Meeting time as % of workday | 35-40% |
| Employees feeling over-meeted | 71% |
| Time in productive meetings | <50% of meeting time |

**Where AI Can Help:**

```
BEFORE MEETING          DURING MEETING           AFTER MEETING
─────────────           ─────────────            ──────────────
• Find time slots       • AI notetaking          • Auto-generate notes
• Generate agenda       • Action item capture    • Distribute summary
• Send invites          • Real-time transcription• Create follow-up tasks
• Prep materials        • Translation            • Track decisions
```

**Time Savings Potential:**

| Task | Manual Time | AI-Assisted Time | Savings |
|------|-------------|------------------|---------|
| Scheduling (finding time) | 10-15 min | 2-3 min | 80% |
| Agenda creation | 10 min | 2 min | 80% |
| Note-taking | 20-30 min | 5 min (review) | 75% |
| Follow-up communications | 15 min | 5 min | 65% |

### Part 2: AI Meeting Workflow (4 minutes)

**Three-Phase AI Meeting System:**

**PHASE 1: PRE-MEETING (Before)**

| Task | AI Tool | How It Works |
|------|---------|--------------|
| Find meeting time | Calendar Gemini panel | "Find 30 min this week for design review" |
| Generate agenda | Gemini in Docs | "Create agenda for 30-min standup with 5 attendees" |
| Prepare materials | Gemini + Drive | "Summarize relevant documents for this meeting" |

**PHASE 2: DURING MEETING**

| Task | AI Tool | How It Works |
|------|---------|--------------|
| Take notes | Meet AI notes | Auto-generated, speaker-labeled |
| Transcribe | Meet transcripts | Full text, searchable |
| Capture actions | Manual + AI assist | Post-meeting extraction |

**PHASE 3: POST-MEETING (After)**

| Task | AI Tool | How It Works |
|------|---------|--------------|
| Generate summary | Gemini from notes | "Summarize key decisions and actions" |
| Draft follow-up | Gemini in Gmail | "Draft email to team with meeting summary" |
| Create task list | Gemini + Tasks | "Extract action items and owners" |

## CONSOLIDATE Phase

### Guided Workshop: Schedule & Generate Agenda (10 minutes)

**Task 1: AI-Assisted Scheduling (3 minutes)**

1. Open Google Calendar
2. Click "Create" or use Gemini panel
3. Try one of these prompts in the description:
   - "Schedule a 30-minute 1:1 with your manager for next week"
   - "Find time for a 45-min team meeting on Tuesday or Wednesday"
4. Send the invite to yourself (or a colleague if in a live training)

**Task 2: Generate Meeting Agenda (4 minutes)**

1. Create new Google Doc titled: "Meeting Agenda - [Your Topic]"
2. Open Gemini (Ctrl+Shift+M)
3. Use this prompt:

```
Create a professional meeting agenda for a 30-minute meeting.

Meeting Type: Weekly Team Sync
Attendees: 6 team members
Purpose: Share updates, identify blockers, coordinate on Q3 goals

Include:
1. Meeting objective (1 sentence)
2. Agenda items with time allocations
3. Pre-meeting reading/prep notes
4. Decision points
5. Parking lot section for off-topic items
```

4. Review and personalize the agenda
5. Save to your Drive

**Task 3: Schedule the Meeting (3 minutes)**

1. Create calendar event with the AI-generated agenda
2. Copy agenda into the event description
3. Set reminder for 15 minutes before
4. Add location/video link
5. Send invite

### AI Notetaking Exercise (3 minutes)

**Simulated Meeting Notes Practice:**

1. Create a new Google Doc titled: "Meeting Notes - [Today's Date]"
2. Use this simulated meeting dialogue:

```
SIMULATED MEETING CONTENT:

Project Status Update - May 27, 2026
Attendees: Alex (PM), Jordan (Design), Taylor (Engineering)

Alex: Q2 launch is on track. We hit 92% of our sprint goals.

Jordan: Design review for new dashboard is complete. Ready for dev handoff.

Taylor: Engineering can start integration Monday. Need final API specs from backend team.

DECISIONS:
- Launch date confirmed: June 15
- Dashboard feature will ship in Phase 2 (July)

ACTION ITEMS:
- Jordan: Send design specs to Taylor by EOD Friday
- Taylor: Schedule backend sync for API review
- Alex: Update stakeholder presentation with new timeline

NEXT MEETING: June 1, 10am
```

3. Paste into Gemini
4. Prompt: "Extract: decisions made, action items with owners, and a 3-bullet summary for those who missed the meeting"

## COMPRESS Phase

### Pattern Recap (1 minute)

**The ONE thing to remember:**

> "AI doesn't replace meeting productivity — it removes the administrative overhead. Use it to capture, not to think."

**Meeting AI Decision Guide:**

```
Use AI for:
• Scheduling coordination (finding times)
• Agenda templating (consistent structure)
• Note capture (post-meeting summary)
• Follow-up drafting (standard communications)

Use human judgment for:
• Meeting necessity (should this meeting exist?)
• Discussion facilitation (leading the conversation)
• Sensitive feedback (relationship-aware communication)
• Strategic decisions (AI can inform, not decide)
```
