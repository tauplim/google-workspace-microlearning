## LEARN Phase

### Part 1: Collaboration Fundamentals (3 minutes)

**The Shift from "Send and Wait" to "Share and Collaborate"**

Traditional workflow:
```
You → Create document → Attach to email → Send → Wait for response → Incorporate changes
```

Modern collaborative workflow:
```
You → Create document → Share link → Colleagues edit simultaneously → AI assists with feedback → Done
```

**Key Collaboration Features in Google Docs:**

| Feature | What It Does | Why It Matters |
|---------|--------------|----------------|
| **Real-time cursors** | See where others are working | Avoids duplicate edits |
| **Suggesting mode** | Track changes without overwriting | Maintains edit history |
| **Comments** | Feedback without changing text | Clarifies intent |
| **Named locations** | Jump to specific sections | Easy navigation |
| **Version history** | See all changes over time | Restore if needed |

**Editing vs. Suggesting Modes:**

| Mode | Who sees changes? | Use when... |
|------|-------------------|-------------|
| **Editing** | Direct changes (no tracking) | You fully own the document |
| **Suggesting** | Proposed changes (tracked) | Reviewing others' work |

### Part 2: AI-Assisted Commenting (4 minutes)

**Four Ways AI Enhances Collaboration:**

1. **Generate Comments** - AI suggests feedback where you might miss issues
2. **Reply to Comments** - AI helps craft diplomatic responses
3. **Resolve Threads** - AI identifies when discussions are complete
4. **Summarize Activity** - AI shows what changed and who did what

**When AI Helps vs. When to Use Human Judgment:**

| AI Helps | Use Human Judgment |
|----------|---------------------|
| Grammar and clarity suggestions | Tone and sensitivity |
| Fact-checking claims | Strategic decisions |
| Summarizing long threads | Relationship nuance |
| Flagging inconsistencies | Creative direction |

### Part 3: Live Demonstration (4 minutes)

**Watch the trainer perform these actions:**

1. **Sharing for Collaboration**
   - Create new document
   - Click Share button
   - Select "Anyone with link can edit"
   - Copy link

2. **Switching to Suggesting Mode**
   - Click dropdown next to Editing in toolbar
   - Select "Suggesting"
   - Type a sentence — notice it appears as green suggestion

3. **Adding an AI-Generated Comment**
   - Highlight existing text
   - Click Comment icon
   - Click Gemini sparkle
   - Prompt: "Suggest a constructive way to improve this sentence"
   - Observe AI-generated feedback

4. **Responding with AI**
   - Click Reply on a comment
   - Click Gemini
   - Prompt: "Help me write a thoughtful acknowledgment"
   - Observe AI draft response

5. **Resolving with AI**
   - Click Resolve on comment thread
   - Show how resolved comments are hidden but accessible

## CONSOLIDATE Phase

### Guided Workshop: Co-Author a Document with AI (10 minutes)

**Task 1: Set Up a Shared Document (2 minutes)**

1. Create new Google Doc titled: "Module 2 Workshop - [Your Name]"
2. Click Share
3. Select "Anyone with the link can edit"
4. Copy the link
5. Send link to your neighbor via chat or email

**Task 2: Practice Suggesting Mode (3 minutes)**

1. Switch to Suggesting mode (dropdown in toolbar)
2. Add a suggestion to your neighbor's document: "Suggest improving one sentence"
3. Wait for neighbor to accept/reject your suggestion
4. Accept or reject one suggestion from your neighbor

**Task 3: AI-Enhanced Comments (5 minutes)**

1. In your document, highlight a paragraph
2. Add a new comment
3. Click the Gemini sparkle icon
4. Use one of these prompts:
   - "Suggest a constructive improvement for this paragraph"
   - "Summarize this section in one sentence"
   - "Identify any unclear statements"

5. Use AI to help you reply to a comment
6. Resolve the comment thread

### Pair Exercise: Peer Review with AI (3 minutes)

**With your partner:**

1. Partner A: Share a short paragraph from your work (3-4 sentences)
2. Partner B: Use AI to generate 2 improvement suggestions
3. Partner B: Reply using AI assistance
4. Both: Discuss - "Did AI help? Was the feedback useful?"

**Discussion:**
- When did AI comments feel helpful?
- When did they feel robotic or inappropriate?

### Share-Back and Debrief (2 minutes)

**Group discussion:**

**Question 1:** What surprised you about AI-generated comments?

**Question 2:** What concerns do you have about using AI in collaborative documents?

## COMPRESS Phase

### Pattern Recap (1 minute)

**The ONE thing to remember:**

> "AI makes collaboration faster but judgment still matters. Use AI for generating options, not making decisions."

**AI Collaboration Decision Tree:**

```
Is the comment about grammar/clarity?
  → YES: AI can likely help
  → NO: Consider human judgment

Is the decision strategic or relational?
  → YES: Use human judgment
  → NO: AI can assist
```
