## LEARN Phase

### Part 1: Capstone Project Brief (3 minutes)

**Your Mission:**

Build a personal **"AI Workspace Playbook"** — a document containing your 5 most valuable AI-powered workflows that you can use daily.

**Requirements:**

| Requirement | Specification |
|-------------|---------------|
| Minimum apps integrated | 3 different Workspace apps |
| AI assistance points | At least 5 Gemini prompts used |
| Workflow variety | At least one from each category: Create, Collaborate, Automate |
| Documentation | Clear enough for a colleague to replicate |
| Real application | Based on actual work you do |

**Deliverable:**
A single Google Doc titled: "[Your Name] AI Workspace Playbook"

---

### Part 2: Workflow Design Methodology (4 minutes)

**The A.P.E. Framework for AI Workflows:**

| Step | Description | Example |
|------|-------------|---------|
| **A**ssess | What trigger or input starts this workflow? | Email arrives, meeting ends, document created |
| **P**rocess | What AI transformations happen? | Summarize, extract, draft, analyze |
| **E**xecute | What is the final output or action? | Email sent, file saved, task created |

**Workflow Design Checklist:**

```
□ What triggers this workflow? (input/event)
□ What Workspace apps are involved? (min 3)
□ What does AI do at each step?
□ What is the final output?
□ How long does this save me?
□ Can a colleague follow this?
```

---

### Part 3: Peer Collaboration Tips (2 minutes)

**During the workshop phase:**

- Help each other troubleshoot
- Share successful prompts
- Give constructive feedback on workflow clarity
- Celebrate diverse approaches

**Feedback Framework:**

| Feedback Type | Example |
|---------------|---------|
| What's clear | "Your workflow steps are easy to follow" |
| What's confusing | "Step 3 could use more detail about what to copy/paste" |
| What's valuable | "I love that you automated the email follow-up" |
| Improvement idea | "Have you considered adding a calendar trigger?" |
