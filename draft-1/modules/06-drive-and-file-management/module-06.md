## LEARN Phase

### Part 1: The File Organization Problem (3 minutes)

**The Typical Drive Scenario**

| Metric | Reality |
|--------|---------|
| Average files per knowledge worker | 10,000+ |
| Time spent searching for files | 2.5 hours/week |
| Files never found | 7% of total |
| Version confusion | 1 in 5 collaborative edits |

**The Cost:** If you spend 2.5 hours/week searching for files at a $50/hour rate, that's $6,500/year in lost productivity.

**Traditional Solutions Fail Because:**
1. **Inconsistent naming** - Everyone has different conventions
2. **Deep folder hierarchies** - Files buried 5+ levels deep
3. **No single source of truth** - Multiple copies of same document
4. **Sharing confusion** - Who has access to what?

**AI Changes the Game:**

Instead of "Where did I save this?" → "What am I looking for?"

### Part 2: AI-Powered Search Capabilities (4 minutes)

**Google Drive Search Evolution:**

| Era | Method | Limitation |
|-----|--------|------------|
| 2010s | File name search | Must know exact name |
| 2020-2024 | Advanced operators | file:doc contains:"budget" |
| 2025+ | Natural language | "Find my Q3 budget from any email attachment" |

**Natural Language Search Examples:**

| What You Say | What Gemini Does |
|--------------|------------------|
| "Find contracts I haven't reviewed yet" | Locates unreviewed contract files |
| "Where is the presentation from last quarter's all-hands?" | Shows presentation with metadata |
| "Find files modified by Sarah in the past week" | Filters by owner and date |
| "Show me all spreadsheets about budget" | Semantic search across content |

**Beyond Search: AI Organization:**

| Feature | What It Does |
|---------|--------------|
| Smart categorization | AI suggests folders based on file content |
| Duplicate detection | Finds files with similar content |
| Access insights | Shows who views and edits your files |
| Storage cleanup | Identifies large/unused files |

### Part 3: Live Demonstration (4 minutes)

**Watch the trainer perform these actions:**

1. **Natural Language Search**
   - Open gemini.google.com or Drive search
   - Type: "Find spreadsheets about monthly expenses"
   - Show semantic results (not just filename matches)

2. **Advanced Search with Filters**
   - Use: "from:me has:attachment older_than:30d"
   - Show filtered results

3. **Create Folder Structure**
   - Create project folder with subfolders:
     ```
     Project Alpha/
     ├── 01-Planning/
     ├── 02-Assets/
     ├── 03-Drafts/
     ├── 04-Review/
     └── 05-Final/
     ```
   - Explain the numbering convention for sort order

4. **Set Sharing Permissions**
   - Show Folder-level vs. File-level sharing
   - Demonstrate "Apply to folder" option
   - Show "Notify stakeholders" feature

5. **Star and Priority Files**
   - Star important files for quick access
   - Create "My Drive" shortcuts

## CONSOLIDATE Phase

### Guided Workshop: Organize a Project Drive (10 minutes)

**Task 1: AI-Assisted File Search (3 minutes)**

1. Open Google Drive or gemini.google.com
2. Try these natural language searches:

```
Search 1: "Find files I haven't opened in over a month"
Result: [How many files? List 3 file names]

Search 2: "Find [topic from your work - e.g., 'marketing materials' or 'project plans']"  
Result: [How accurate was the semantic search?]

Search 3: "Show files modified this week"
Result: [How many files did you find?]
```

**Task 2: Create Project Folder Structure (4 minutes)**

1. Create a new folder called: "Module 6 Project - [Your Project Idea]"
2. Create these subfolders inside:
   ```
   [Your Project]/
   ├── 01-Background/
   ├── 02-Materials/
   ├── 03-Drafts/
   ├── 04-Feedback/
   └── 05-Archive/
   ```
3. Add a brief description to each folder
4. Move at least one file into the structure

**Task 3: Set Sharing Permissions (3 minutes)**

1. Share the main project folder with a partner (if in training)
2. Choose "Anyone with link can view"
3. For a subfolder inside, choose "Restricted - only you"
4. Note: Why might you want different permissions at folder vs. file level?

### Search Challenge Exercise (3 minutes)

**Can you find it using AI?**

Work with a partner to test AI search accuracy:

1. Partner A: Create a file with a descriptive name but obscure content (e.g., "Q3 Analysis.docx" containing text about summer sales trends)
2. Partner B: Search using natural language that should match the content
3. Did AI find it? How quickly?

**Test these search phrases:**
- "Find the document about [partner's topic]"
- "Show me files related to [industry term]"

## COMPRESS Phase

### Pattern Recap (1 minute)

**The ONE thing to remember:**

> "AI search means 'what' over 'where.' But good folder structure still helps when AI isn't available."

**Drive Organization Principles:**

| Principle | Practice |
|-----------|----------|
| Shallow hierarchy | Max 3-4 levels deep |
| Numbered folders | 01, 02, 03 for consistent sorting |
| Descriptive names | Include dates: "Q3 Report 2026-07" |
| Consistent structure | Same folder pattern across projects |
| Archive regularly | Move completed projects to Archive |
