**Version:** Draft 1

**Format:** Trainer-led, 40:60 lecture:workshop split
**Framework:** Learn, Consolidate, Compress

---

## Pedagogical Framework: Learn, Consolidate, Compress

### The "Learn, Consolidate, Compress" Model

This course uses a cognitive compression methodology designed for adult professionals who need to acquire practical skills with minimal time investment while maximizing long-term retention.

| Phase | Focus | Duration | Activities |
|-------|-------|----------|------------|
| **Learn** | New concept introduction | 12 min (40%) | Direct instruction, live demos, conceptual anchoring |
| **Consolidate** | Hands-on practice | 18 min (60%) | Guided workshops, peer exercises, real-world application |
| **Compress** | Memory encoding & pattern recognition | Embedded throughout | Spaced repetition, workflow automation, muscle memory building |

```
├──────────────┼──────────────────────────────────────────────┤
│ 0:12 - 0:27  │ CONSOLIDATE PHASE (15 min)                  │
│              │ • Guided workshop (10 min)                   │
│              │ • Peer exercise / discussion (3 min)         │
│              │ • Share-back & debrief (2 min)               │
├──────────────┼──────────────────────────────────────────────┤
│ 0:27 - 0:30  │ COMPRESS & CLOSE                             │
│              │ • Pattern recap (1 min)                     │
│              │ • Take-away automation/shorcut (1 min)      │
│              │ • Bridge to next session (1 min)            │
└─────────────────────────────────────────────────────────────┘
```

### Microlearning Design Principles

1. **One Core Skill Per Session** - Each 30-minute session targets ONE specific competency
2. **Zero-Buffer Retention** - Every workshop task is a directly replicable workflow
3. **AI as Co-pilot** - Gemini is positioned as learning accelerant, not feature tutorial
4. **Compression Artifacts** - Each session produces a "cheat sheet" automation or shortcut
5. **Interleaving** - Later modules intentionally revisit and strengthen earlier concepts

## Module Specifications

### Module 1: Foundation - Google Workspace with AI
**Duration:** 30 minutes | **Prerequisites:** None

| Phase | Time | Content |
|-------|------|---------|
| Learn | 12 min | Ecosystem overview, Gemini integration points, AI-assisted workflows |
| Consolidate | 18 min | Navigate Workspace apps, locate Gemini in each app, first AI prompt |

**Success Criteria:** Participant can describe the 5 core Workspace apps and initiate Gemini in at least 3 of them.

**Compression Artifact:** "Gemini Hotkeys" quick reference card

---

### Module 2: Collaboration with AI
**Duration:** 30 minutes | **Prerequisites:** Module 1

| Phase | Time | Content |
|-------|------|---------|
| Learn | 12 min | Real-time collaboration patterns, AI-assisted commenting, smart reply |
| Consolidate | 18 min | Co-author a document using Gemini suggestions, resolve comments with AI |

**Success Criteria:** Participant can co-edit a document and use AI to generate, respond to, or resolve comments.

**Compression Artifact:** "AI Collaboration Checklist" workflow

---

### Module 3: Document Automation
**Duration:** 30 minutes | **Prerequisites:** Module 1, 2

| Phase | Time | Content |
|-------|------|---------|
| Learn | 12 min | Template creation, Gemini for drafting, version history with AI |
| Consolidate | 18 min | Create a templated document, use AI to generate 3 variations |

**Success Criteria:** Participant can create a reusable template and use Gemini to draft content variations.

**Compression Artifact:** "Document Automation Template" starter kit

---

### Module 4: AI-Powered Research
**Duration:** 30 minutes | **Prerequisites:** Module 1-3

| Phase | Time | Content |
|-------|------|---------|
| Learn | 12 min | Gemini for information synthesis, source citation, data extraction |
| Consolidate | 18 min | Research a topic using Gemini, extract key points, generate summary |

**Success Criteria:** Participant can use Gemini to synthesize information from multiple sources into a structured summary.

**Compression Artifact:** "Research Workflow" automation sequence

---

### Module 5: Meetings & Calendaring
**Duration:** 30 minutes | **Prerequisites:** Module 1

| Phase | Time | Content |
|-------|------|---------|
| Learn | 12 min | Meet scheduling with Gemini, agenda generation, AI notetaking |
| Consolidate | 18 min | Schedule a meeting, generate agenda, use AI to take and distribute notes |

**Success Criteria:** Participant can schedule a meeting and generate meeting notes using Gemini.

**Compression Artifact:** "Meeting Automation" script

---

### Module 6: Drive & File Management
**Duration:** 30 minutes | **Prerequisites:** Module 1

| Phase | Time | Content |
|-------|------|---------|
| Learn | 12 min | AI-powered search, auto-organization, sharing workflows |
| Consolidate | 18 min | Organize Drive with AI assistance, create shared folder structure |

**Success Criteria:** Participant can use AI search to locate files and implement an organized Drive structure.

**Compression Artifact:** "Drive Organization System" template

---

### Module 7: Automation with Gemini
**Duration:** 30 minutes | **Prerequisites:** Module 1-6

| Phase | Time | Content |
|-------|------|---------|
| Learn | 12 min | Gemini for repetitive tasks, bulk operations, workflow chains |
| Consolidate | 18 min | Automate 3 repetitive tasks using Gemini prompt patterns |

**Success Criteria:** Participant can identify automation opportunities and execute a Gemini prompt chain.

**Compression Artifact:** "Automation Playbook" with 10 prompt patterns

---

### Module 8: AI Integration Final Project
**Duration:** 30 minutes | **Prerequisites:** All previous modules

| Phase | Time | Content |
|-------|------|---------|
| Learn | 12 min | Capstone brief, project workflow, peer collaboration |
| Consolidate | 18 min | Execute capstone project, peer review, course wrap-up |

**Success Criteria:** Participant delivers a completed workflow that integrates at least 3 Google Workspace apps with Gemini.

**Compression Artifact:** Personal "AI Workspace Playbook" custom template
