# Google Workspace with AI - Microlearning Course
## Draft 1 - Complete Package

---

## Course Overview

**Trainer-Led Microlearning Course for Beginners**

| Aspect | Details |
|--------|---------|
| **Duration** | 8 modules × 30 minutes = 4 hours total |
| **Format** | Trainer-led, 40:60 lecture:workshop split |
| **Framework** | Learn, Consolidate, Compress |
| **Audience** | Beginners with basic computer literacy |
| **Max Participants** | 12 (for workshop quality) |

---

## Course Structure

### The "Learn, Consolidate, Compress" Framework

Each 30-minute session follows this cognitive compression methodology:

| Phase | Duration | Activities |
|-------|----------|------------|
| **Learn** | 12 min (40%) | Concept introduction, live demo, key principles |
| **Consolidate** | 18 min (60%) | Guided workshop, peer exercise, share-back |
| **Compress** | Embedded | Memory encoding, pattern recognition, take-away artifacts |

## Facilitation Notes

### Trainer Preparation
- [ ] Test Google Workspace demo accounts
- [ ] Verify Gemini AI features enabled
- [ ] Pre-create shared Drive folder structure
- [ ] Print participant workbooks
- [ ] Set visible timer

### Session Timing
- Use visible timer for phase management
- 40:60 split is strict — cut lecture if running long
- Compression artifacts should take < 60 seconds

### Common Adjustments
- **Fast group:** Skip to challenge extension tasks
- **Slow group:** Reduce to 2-3 workshop tasks, extend share-back
- **Virtual:** Use breakout rooms for pair exercises

## Research-Backed Design

This course incorporates evidence-based adult learning principles:

| Principle | Implementation |
|-----------|----------------|
| **Spaced repetition** | Module 4 revisits Module 3 concepts; Module 7 interleaves earlier skills |
| **Chunking** | One core skill per 30-min session |
| **Active recall** | Peer share-back in every Consolidate phase |
| **Cognitive load management** | 40:60 split reduces passive listening |
| **Authentic assessment** | Capstone project uses real work scenarios |
| **Social learning** | Pair exercises and peer review built into every module |
